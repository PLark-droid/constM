# 職人向け操作ガイド

スマホから出退勤と日報を出す方法をまとめたガイドです。

---

## はじめに -- Larkアプリの準備

## よく使う言葉

| 言葉 | 意味 |
|------|------|
| 打刻（だこく） | 出勤・退勤の時刻を記録すること |
| 承認（しょうにん） | 班長や監督が「内容OK」と確認すること |
| 差戻し（さしもどし） | 日報の内容に修正が必要な時、書き直しを依頼されること |
| 人工（にんく） | 1日分の作業量の単位。丸1日働くと1.0人工 |
| 日報（にっぽう） | 今日やった作業の報告 |

### アプリを入れる

1. スマホで「App Store」（iPhone）または「Google Play」（Android）を開いてください
2. 「Lark」と検索してください
3. インストールしてください
4. 会社から渡されたアカウントでログインしてください

### スマホの設定

1. Larkの通知を **ON** にしてください
2. 位置情報を **「アプリ使用中に許可」** にしてください

```
┌─────────────────────────────┐
│  設定 > Lark > 位置情報      │
│                             │
│  (  ) 許可しない            │
│  (*) アプリ使用中に許可  <-- ここ
│  (  ) 常に許可              │
└─────────────────────────────┘
```

### フォームをブックマークに登録する

班長または管理者からQRコードを受け取ってください。
3つのフォームを登録します。

```
  登録するフォーム:

  [1] 出勤打刻    ... 朝、現場に着いたら使う
  [2] 退勤打刻    ... 夕方、帰る前に使う
  [3] 日報提出    ... 夕方、その日の作業を報告する
```

**登録の手順:**

1. QRコードをスマホのカメラで読み取ってください
2. Larkアプリでフォームが開きます
3. 右上の「ブックマーク（星マーク）」をタップしてください
4. 3つのフォーム全てで同じことをしてください

## 1. 出勤打刻

**いつ:** 朝、現場に着いたらすぐ

### 手順

```
  +---------------------------------------+
  |  ステップ          やること            |
  +---------------------------------------+
  |  (1)  Larkアプリを開く                 |
  |  (2)  ブックマークから「出勤打刻」を開く |
  |  (3)  自分の名前を選ぶ                 |
  |  (4)  今日の現場を選ぶ                 |
  |  (5)  時刻を確認する（自動で入ります）   |
  |  (6)  「送信」をタップ                 |
  +---------------------------------------+
        完了!
```

│  └────────────────────┘  │
│                          │
│  現場 *                  │
│  ┌────────────────────┐  │
│  │ 現場を選ぶ      ▼  │  │

### よくある質問

| 困ったこと | どうする？ |
|-----------|----------|
| 時刻を間違えた | 班長に伝えてください |
| 現場に直行した | 「出勤ステータス」を「直行」に変えてください |
| 現場が出てこない | 班長に連絡してください |
| 送信ボタンが押せない | 名前と現場が選ばれているか確認してください |

## 3. 日報提出

  |  (3)  自分の名前を選ぶ                    |

  |  (6)  その作業の時間を入れる               |

### 日報の書き方のコツ

**進捗メモ -- 場所 + 作業 + どこまで終わったか**

```
  書き方:

    良い例:
    ・「3F 鉄筋組立 80%完了」
    ・「外壁タイル貼り 南面 完了」
    ・「1F配管 トイレ周り 50%」

    短くてOK:
    ・「鉄筋組立 完了」
    ・「型枠ばらし 半分」
```

**安全で気になったこと -- 素直に書いてください**

```
  書き方:

    ・「3F開口部の手すりがグラついていた」
    ・「足場の板が1枚外れかけていた」
    ・「雨で通路が滑りやすかった」
    ・「特になし」でもOK
```

**写真 -- あると助かります**

```
  撮るとよいもの:

    ・作業が終わった場所
    ・問題があった箇所
    ・危険だと感じた場所
```

### 作業時間の入れ方

0.5時間きざみで入れてください。

```
  よくある例:

    作業が1つの日:
      作業1 = 7.0 時間

    作業が2つの日:
      作業1 = 4.0 時間
      作業2 = 3.0 時間

    作業が3つの日:
      作業1 = 3.0 時間
      作業2 = 2.5 時間
      作業3 = 1.5 時間
```

## 4. 通知が届いたら

### 日報リマインダー（18:00ごろ届きます）

```
  ┌──────────────────────────┐
  │  Lark通知               │
  │                         │
  │  「日報がまだ提出されて   │
  │    いません」            │
  │                         │
  └──────────────────────────┘

  --> 日報をまだ出していない場合に届きます。
      日報を提出してください。
```

### 差戻し通知

```
  ┌──────────────────────────┐
  │  Lark通知               │
  │                         │
  │  「日報が差戻しされ      │
  │    ました」             │
  │                         │
  └──────────────────────────┘

  --> 日報の内容に直しが必要です。
      コメントを確認して、直して再提出してください。
```

## 5. こんな時はどうする？

| 困ったこと | どうすればよいか |
|-----------|----------------|
| スマホが壊れた | 班長に伝えてください。代わりに入力してもらえます |
| パスワードを忘れた | 管理者に連絡してください |
| フォームが開けない | Larkアプリを最新に更新してください |
| 間違えて送信した | 班長に伝えてください。直してもらえます |
| 2つの現場に行った日 | 現場ごとに出勤・退勤を打刻してください |
| 送信ボタンを何度も押した | 班長に伝えてください。余分なデータを消してもらえます |
| 昨日の分を出し忘れた | 日報の「日付」を昨日に変えて提出してください |

## 1日の流れ

```
    朝                  夕方               夜
    |                   |                  |
    v                   v                  v

  現場到着            作業終了           帰宅後
    |                   |                  |
    v                   v                  v

  +---------+       +---------+       +---------+
  | 出勤打刻 |       | 退勤打刻 |       |（日報が  |
  | (10秒)  |       | (10秒)  |       |  まだなら |
  +---------+       +---------+       |  提出）  |
                        |              +---------+
                        v
                    +---------+
                    | 日報提出 |
                    | (1〜2分) |
                    +---------+

  ※ 日報は退勤打刻のあとすぐがベストです
  ※ 遅くても 18:00 までに出してください
```
